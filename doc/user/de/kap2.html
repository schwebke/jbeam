Kapitel 2

Einstellungen

Querschnittswerte

Wenn neue Elemente eingefügt werden, so erhalten diese Standardwerte für die Querschnittsgrössen (z.B. EA und EI).

Diese Vorgabewerte können unter 'Preferences - sectional vals' eingestellt werden.

Visuelles Raster und Fangraster

Der Zeichenbereich ist zur Orientierung mit einem Punktraster hinterlegt. Den Abstand zwischen zwei Punkten stellt man unter 'Preferences - snap and grid' im Feld 'visible grid' ein. Dieses Raster hat keinen Einfluß auf das Anlegen von Objekten.

Im Feld 'snap grid' wird der Abstand des (nicht sichtbaren) Fangrasters angezeigt. Neue Objekte können nur auf Punkte des Fangrasters gelegt werden. Zur Kontrolle wird in der Koordinatenanzeige der nächstliegende Fangrasterpunkt angezeigt.

Um das visuelle Raster oder das Fangraster zu deaktivieren, gibt man in das entsprechende Feld 0 ein.

Im Feld 'check radius' gibt man den Toleranzradius in Pixeln an, der bei der Selektion von Objekten gilt. Ein höherer Wert erlaubt somit auch dann das Selektieren, wenn das Objekt weiter verfehlt wird.

Anzeigegröße

Im Dialog 'Preferences - view' kann die Größe der angezeigten Objekte und Kurven konfiguriert werden.

Unter 'base size' gibt man die Größe einer Basiseinheit in Pixeln an. Damit wird u.a. die Darstellungsgröße von Knoten, Elementen und Kräften beeinflusst.

In den '... scale'-Feldern wird die Überhöhung der Ergebnisplots eingestellt:

displacement scaleVerschiebungsfigur
moment scaleMomentenlinie
normal force scaleNormalkraftlinie
shear force scaleQuerkraftlinie

Zoom

Mittels bzw. 'Preferences - zoom in' wird der Zeichenbereich vergrößert, mit bzw. 'Preferences - zoom out' verkleinert.

bzw. 'Preferences - show all' wird die gesamte Struktur formatfüllend angezeigt.

bzw. 'Preferences - zoom window' zoomt einen rechteckigen Ausschnitt.

bzw. 'Preferences - pan' verschiebt den dargestellten Ausschnitt.

Durch ziehen mit der mittleren Maustaste und mit dem Mausrad kann ebenfalls der Ausschnitt verschoben bzw. vergrößert oder verkleiner werden.
