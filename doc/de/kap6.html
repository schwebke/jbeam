Kapitel 6

Ergebnisausgabe

Mit 'Results - calculate static' wird das System neu berechnet. Dies ist nach jeder Änderung nötig, wenn Ergebnisse angezeigt werden sollen. Falls das System kinematisch ist, wird eine entsprechende Fehlermeldung ausgegeben, oder es kommt zu unrealistisch hohen Verschiebungen, wenn die Singularität der Steifigkeitsmatrix aufgrund von Rundungsfehlern nicht erkannt werden kann.

Mit show ...-Menücheckboxen können die entsprechenden Kurven aktiviert werden. Es wird keine automatische Skalierung vorgenommen, bei Bedarf ist die Größe wie unter Anzeigegröße beschrieben anzupassen.

Mittels 'Results - write ASCII-file' bzw. 'Results - write HTML-file' kann das numerische Ergebnis der Rechnung in einer Datei abgelegt werden. Dabei werden nur Objekte berücksichtigt, die ein Label erhalten haben.

Neben der Ausgabe in eine Datei kann nach Wechsel in den 'Info'-Modus () durch Anklicken der Objekte ein Dialog mit den Zahlenwerten der Ergebnisse angezeigt werden. Die Objekte müssen hierzu nicht mit einem Label versehen sein.

Da Kraft- und Verschiebungsgrößen nur an den Knoten und Elementenden ausgegeben werden, sind bei Bedarf an den interresanten Stellen zusätzliche gelabelte Knoten einzufügen, auch wenn dies zur Modellierung des Systems nicht unbedingt erforderlich wäre.
